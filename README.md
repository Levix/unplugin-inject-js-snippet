# unplugin-inject-js-snippet

Support for injecting js snippet into html and js files.

Starter template for [unplugin](https://github.com/unjs/unplugin).

## Install

```bash
npm i unplugin-inject-js-snippet
```

<details>
<summary>Vite</summary><br>

```ts
// vite.config.ts
import Starter from 'unplugin-inject-js-snippet/vite'

export default defineConfig({
  plugins: [
    Starter({ /* options */ }),
  ],
})
```

Example: [`playground/`](./playground/)

<details>

```ts
// webpack.config.js
module.exports = {
  /* ... */
```

<details>
<summary>Nuxt</summary><br>

```ts
// nuxt.config.js
export default {
  buildModules: [
    ['unplugin-inject-js-snippet/nuxt', { /* options */ }],
  ],
}
```

> This module works for both Nuxt 2 and [Nuxt Vite](https://github.com/nuxt/vite)

<br></details>
